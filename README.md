### 🎓 AI Placement Preparation Portal
> A full-stack placement portal built with **Node.js + MongoDB + Express.js**

| Feature | Description |
|:---|:---|
| 📝 **TCS iON Exam Console** | Real exam-style timer, navigator & question states |
| 💻 **Technical MCQs** | C, C++, Java, DBMS, OS, Networks |
| 🏢 **Placement Drives** | Browse companies, eligibility, packages & apply |
| 📄 **Resume Builder** | Auto-generate professional PDF from profile |
| 🏆 **Leaderboard** | Compete with peers, track rankings |
| 🤖 **AI Career Guide** | Roadmaps for 7+ tech career paths |
| 🔒 **Security** | Helmet, rate limiting, bcrypt, session hardening |
| 🏗️ **MVC Architecture** | Controllers, routes, middleware, models separated |
